# 4 Context-free grammars 

A context-free grammar (CFG) is a 4-tuple G = ( V, $\Sigma$ , R, S ) where: 

1. V is a finite set of variables 

2. $\Sigma$ is a finite set of terminals such that V $\cap$ $\Sigma$ = $\emptyset$ . 

3. R is a finite set of production rules such that each rule is an assignment of a string of variables and terminals to a single variable. 

4. S is the start variable 

The CGF G1 = ( {A}, {x, y, $\varepsilon$}, R, A ) where R is given by: 

* A -> xAy
* A -> AA
* A -> $\varepsilon$

Some of the strings generated by G1 include: 

* xy
* xxxyyy
* xyxy
* xxyyxxyy

## 4.1 shorthand for production rules 

Instead of writing each element in the set of R we can use logical or “ | ” to write the rules more compactly: 

$$A \to xAy | AA | \varepsilon$$

## 4.2 what’s the context? 

By assigning a string of variables and terminals to a single variable we imply that the context under which the production rule is applied is not important; hence the name “context-free”. For example the production rule xxAy $\to$ xAy is not context-free. 

## 4.3 derivations 

The derivation of a string generated a CFG is the sequence of substitutions used to generate it. For example, the string xxxyyy generated by G 1 has derivation: 

A $\Rightarrow$ xAy $\Rightarrow$ xxAyy $\Rightarrow$ xxxAyyy $\Rightarrow$ xxxεyyy

## 4.5 language of a grammar 

Let u $\Rightarrow$^* v represent a sequence of production rules in R such that v can be derived from u , where u $\in$ V and v $\in$ $\Sigma$. Then the language of a grammar G = ( V, $\Sigma$ , R, S ) is given by: 

$L(G)$ = {w $\in$ $\Sigma$* | S $\Rightarrow$^* w}
